## BookList App

A simple web application to manage a list of books with their titles, authors, and publication years. This project was developed using HTML, CSS, and JavaScript.

## Features

- Add a book with title, author, and year.
- Display the list of added books.
- Responsive design with a modern user interface.

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository: git clone https://github.com/riteshpaarihar/booklist-app.git
2. Navigate to the project directory:cd booklist-app
3. Open index.html in your web browser: open index.html
